The number in the left box is the Magic Number and a square is formed with n*n square where n is the number in the 2nd box.
This program is implemented using Genetic Algorithm and is endeavoured to find a number sequence such that each row,column and diagonal sums up to our desired Number.
Green strip represents a line summing up to desired number and Red strip denotes that it is not yet done.
Change the values on the boxes and press Run.
